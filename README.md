Employee Attrition Dashboard
The Employee Attrition Dashboard provides a comprehensive view of workforce attrition patterns within the organization. It helps identify trends in employee turnover across departments, job roles,salary etc enabling HR and management to take proactive measures.The dataset for this analysis has been sourced from Kaggle.
The Dashboard is built using the following tools and technologies :
Power BI Desktop : The main data visualization platform used for dashboard creation
Power Query : used for Data cleaning and Data Transformation
DAX : used for calculated measures and logics
Data Modeling : Relationships established among tables
